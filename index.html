Steve's Encode64 Utility

My encode64 tool used for creating data URIs

Download v1.2 Stand-alone executable for Windows

This is my super-simple Encode64 tool for Microsoft Windows. It's used for creating data URIs.

Usage

  • Click the Select File button, select the file you want to process. The tool will show the encoded string.
  • Then click the Copy to Clipboard button. Paste the string wherever you need it.
